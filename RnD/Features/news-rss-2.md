## SECTION 1: FREE NEWS APIs (RECOMMENDED FOR DEVELOPERS)

### 1. **GNews API** 
- **URL:** https://gnews.io
- **Coverage:** 80,000+ sources, 41 languages, 71 countries
- **Free Tier:** 100 requests/day, 30-day history, 12-hour delay
- **Historical Data:** 2020-present
- **Response Format:** JSON
- **Best For:** Global English-language news aggregation

### 2. **NewsData.io API**
- **URL:** https://newsdata.io
- **Coverage:** 50,000+ sources, 88 countries, 22 languages
- **Free Tier:** Substantial free access (check website for current limits)
- **Historical Data:** 6+ years available
- **Response Format:** JSON
- **Best For:** Multilingual news, heavy language support

### 3. **Currents News API**
- **URL:** https://currentsapi.services/en
- **Coverage:** 70+ countries, 18+ languages, 43,500+ domains
- **Free Tier:** 1,000 requests/day (no credit card required)
- **Historical Data:** 4 years
- **Response Format:** JSON
- **Best For:** Cost-effective, real-time updates, diverse languages

### 4. **NewsAPI.org**
- **URL:** https://newsapi.org
- **Coverage:** 150,000+ sources globally
- **Free Tier:** 100 requests/day, 1-month historical data
- **Historical Data:** 1 month lookback
- **Response Format:** JSON
- **Best For:** Breaking news, easy integration, broad coverage

### 5. **Event Registry API**
- **URL:** https://eventregistry.org
- **Coverage:** 150,000+ sources, 60+ languages
- **Free Tier:** Limited; contact for pricing
- **Features:** AI event detection, sentiment analysis, entity extraction
- **Response Format:** JSON
- **Best For:** Event-based news tracking, trend analysis

### 6. **Mediastack API**
- **URL:** https://mediastack.com
- **Coverage:** 7,500+ sources globally
- **Free Tier:** Substantial free tier available
- **Historical Data:** Real-time updates, 1-minute refresh
- **Response Format:** JSON
- **Best For:** Live news data, real-time streaming

### 7. **World News API**
- **URL:** https://worldnewsapi.com
- **Coverage:** 86+ languages, 210+ countries
- **Features:** Sentiment analysis (AI-based), MCP protocol support
- **Response Format:** JSON
- **Best For:** Comprehensive global coverage, AI analysis

### Coverage Bias Awareness
- **Conservative sources:** FOX News, National Review, Gateway Pundit, Breitbart
- **Liberal sources:** Daily Kos, Intercept, Mother Jones, MSNBC
- **Centrist sources:** Reuters, AP News, BBC, Financial Times, The Economist
- **International perspectives:** Al Jazeera (Qatar), RT/TASS (Russia), ECNS (China), DW (Germany)

### Data Privacy
- Check API terms of service for commercial use restrictions
- Some APIs require attribution for republished content
- Images and full article text may have additional licensing requirements

- GNews: 100 requests/day
- NewsAPI: 100 requests/day
- Currents: 1,000 requests/day
- World News: Check website for current limits

### Historical Data Availability
